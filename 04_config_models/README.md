# 第四章：配置体系与模型接入基础

本章从“系统可控”的角度讲配置与模型：先理解配置文件结构与优先级，再完成模型供应商接入，最后建立模型选择与故障转移的基础策略。读完本章后，读者应能独立回答三个问题：当前配置从哪里生效、当前模型为什么被选中、当前失败时系统会如何退化。

**本章学习目标**

- 理解 openclaw.json 的核心结构与配置优先级。
- 掌握模型供应商接入流程与认证方式。
- 建立模型选择的质量/成本/延迟/可靠性四维决策框架。
- 配置并验证基础故障转移链路。
