# 第十三章：附录 - 参考资料与速查

附录提供面向实战的高频参考内容，包括术语表、配置模板、故障排查检查单、API 参考与延伸阅读。这些内容并不需要从头到尾阅读，而是应该在日常开发与运维中作为快速查阅的工具。

## 本章内容导读

本章包括以下几个小节：

- **[术语表](_glossary.md)**：OpenClaw 全书涉及的核心术语与概念的统一解释。
- **[配置模板与样例](_config_templates.md)**：不同场景下的 openclaw.json 配置模板，开箱即用。
- **[故障排查检查单](_troubleshooting_checklist.md)**：遇到问题时的系统化检查流程。
- **[API 与 SDK 参考](_api_reference.md)**：OpenClaw 提供的 API 接口与 SDK 的完整参考。
- **[延伸阅读与参考资料](_reading_list.md)**：深入学习的推荐资源与论文。
- **[命令速查手册](_command_cheatsheet.md)**：常用 CLI 命令的快速参考。

## 使用建议

- **开发阶段**：重点参考配置模板与命令速查，加快开发效率。
- **调试阶段**：使用故障排查检查单与 API 参考，定位问题。
- **生产运维**：参考术语表与延伸阅读，建立团队共识。
